## 📄 الملف الأول: `sample_videos_30.csv`

### ماذا يحتوي؟
**30 فيديو نموذجي** بصيغة CSV مع محتويات عربية حقيقية.

### كيفية الاستخدام:

```
1. افتح الموقع: http://localhost:8080
2. اذهب إلى "معالج الملفات" (📁) من الشريط الجانبي
3. اضغط على منطقة الرفع أو اسحب الملف
4. اختر: sample_videos_30.csv
5. اضغط "🚀 بدء التحليل"
6. انتظر النتائج (حوالي 2 دقيقة)
7. اضغط "تصدير النتائج" لحفظ الملف
```

### محتوى الملف:
```csv
title,description,keywords,concept,contentType
صباح هادئ - فن القهوة,مشهد سينمائي للقهوة...,mindfulness,morning ritual,...,طقوس صباحية,video
الطبخ التقليدي - الخبز الحرفي,وثائقي قصير عن فن صنع الخبز...,artisan bread,...,صنع الخبز,video
...
```

### ماذا تتوقع؟
✅ **النتائج:**
- معظم الفيديوهات ستحصل على 60-85% قبول
- بعضها سيكون احتمال قبوله 75%+
- ستظهر توصيات محددة لكل واحد

| أبسط للقراءة | أفضل للبيانات المعقدة |

## 🔧 إذا أردت تعديل الملفات

### لإنشاء ملف CSV خاص بك:

```
1. افتح Notepad أو Excel
2. أنشئ أعمدة:
   • title (العنوان)
   • description (الوصف)
   • keywords (الكلمات المفتاحية)
   • concept (المفهوم)
   • contentType (نوع المحتوى)
3. املأ البيانات
4. احفظ باسم "my_videos.csv"
5. رفع إلى معالج الملفات
```

### لإنشاء ملف JSON خاص بك:

```
1. استخدم موقع مثل jsoneditor.io
2. انسخ هيكل sample_videos_30.json
3. عدّل البيانات
4. احفظ
5. رفع إلى معالج الملفات
```
