**معالج الملفات** يسمح لك برفع ملف يحتوي على **100+ فيديو** وتحليلها جميعاً **تلقائياً** في دقائق!

## 🎯 كيفية الاستخدام

### الخطوة 1: افتح معالج الملفات
```
من الشريط الجانبي (يسار الشاشة)
اختر: "معالج الملفات" 📁
```

### الخطوة 2: جهّز ملف البيانات

اختر أحد الصيغتين:

#### **خيار 1: ملف CSV**

```csv
title,description,keywords,concept,contentType

```

#### **خيار 2: ملف JSON**

```json
[
  {

  {
    "title": "فيديو طهي تقليدي",
    "description": "وثائقي قصير عن الطهي التقليدي...",
    "keywords": "traditional cooking, heritage recipe",
    "concept": "طهي",
    "contentType": "video"
  }
]
```

## 📝 بنية الملف

### الحقول المطلوبة:

| الحقل | الوصف | مثال |
|------|-------|------|
| **title** | عنوان الفيديو/الصورة | "صباح هادئ - فن القهوة" |
| **description** | وصف مفصل | "مشهد سينمائي للقهوة الصباحية..." |
| **keywords** | كلمات مفتاحية (مفصولة بفاصلة) | "mindfulness, morning, coffee" |
| **concept** | المفهوم الأساسي | "طقوس صباحية" |
| **contentType** | نوع المحتوى | "video" أو "image" |

## ⚠️ أخطاء شائعة

### ❌ الخطأ #1: ملف بصيغة خاطئة
```
لا تنسَ: CSV أو JSON فقط!
لا XLSX أو PDF أو صور
```

### ❌ الخطأ #2: حقول ناقصة
```
تأكد من وجود كل الحقول:
title, description, keywords, concept, contentType
```

### ❌ الخطأ #3: فواصل خاطئة
```
في CSV: استخدم فاصلة (,)
في keywords: استخدم فاصلة مع مسافة (, )
```

### ❌ الخطأ #4: نص فارغ
```
اتركه فارغاً أو قيمة افتراضية
لا تترك حقل بدون قيمة
```
